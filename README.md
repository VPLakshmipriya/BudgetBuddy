# BUDGET BUDDY
"No more ‘Where did my money go?’ moments."

A fun and interactive expense tracker web application that helps users track their daily spending with visual charts and dark mode support.

---

## 🚀 Features

- ➕ Add expenses
- 🗑 Delete expenses
- 📊 Animated Pie Chart (Category wise)
- 📅 Monthly Comparison Bar Chart
- 🌙 Dark / Light Mode
- 💾 Data stored using LocalStorage

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js
